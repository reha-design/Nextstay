# ✅ 2026-03-06 프로젝트 킥오프 및 기초 설계 완료 리포트

넥스트스테이(Nextstay) 프로젝트의 첫 날로, 전체적인 아키텍처 비전 수립과 핵심 API 명세 작성을 완료하였습니다.

## 🛠️ 작업 내용

### 1. 프로젝트 비전 및 테크 스택 확정
-  **목표**  : Kotlin/Spring Boot와 최신 프론트엔드 기술을 활용한 고성능 숙박 예약 시스템 구축.
-  **테크 스택**  :
  -  **Backend**  : Kotlin + Spring Boot (핵심 비즈니스), Bun + Elysia.js (부가 서비스)
  -  **Frontend**  : Vue.js (Nuxt 3), Pinia, SCSS
  -  **Infrastructure**  : MySQL, RabbitMQ, Redis, AWS S3, CloudFront

### 2. API 엔드포인트 초안 작성 (`api_endpoints.md`)
-  **핵심 도메인 설계**  :
  -  **숙소 & 검색**  : 전체 목록 조회 및 지도 기반 필터링 API 설계.
  -  **예약 & 결제**  : 180일 영업 규제 한도 체크 로직을 포함한 예약 요청 Flow 설계.
  -  **유저 & 인증**  : JWT 기반 GUEST/HOST 권한 분할 및 통합 회원가입 flow.
  -  **B2B 백오피스**  : 사장님 전용 정산, 점검 신청, CS 티켓 관리 API 설계.

### 3. 시스템 아키텍처 및 흐름 설계
-  **데이터 흐름**  : 사용자 요청부터 DB 트랜잭션, 그리고 RabbitMQ를 통한 비동기 이벤트 전파 구조 설계.
-  **문서화**  : `architecture_rabbitmq.md` 등을 통해 메시지 큐 기반의 확장성 있는 구조 시각화 준비.

## 🔍 주요 성과
- 프로젝트의 전체적인 청사진(Blueprint) 완성.
- 백엔드 개발을 위한 구체적인 API 명세서 확정.
- 복잡한 B2B 비즈니스 요구사항(Lazy Issue 쿠폰, 180일 규제 등)에 대한 기술적 해결 방안 수립.
