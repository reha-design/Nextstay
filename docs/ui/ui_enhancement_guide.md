# 미스터멘션 포트폴리오 UI/CSS 고도화 가이드

본 문서는 동적 가격 엔진 도입으로 인해 추가된 데이터들이 사용자에게 최상의 경험(UX)을 줄 수 있도록 스타일을 보정하기 위한 **수정 프롬프트**입니다.

---

## 🚀 수정 프롬프트 1: 메인 페이지 카드 (RoomCard.vue) 리뉴얼

**[목표]** 1박 평균가와 총 예약 금액의 대비를 통해 '장기 숙박 시의 경제성'을 극대화하여 어필합니다.

### 🎨 스타일 상세 지침

1.  **가로 2분할 레이아웃**: `pricing-container`를 `flex`로 구성하고 중앙에 연한 회색(`#EEEEEE`)의 세로 구분선을 배치합니다.
2.  **좌측 (1박 평균)**:
    - 숫자 크기를 키우고(`20px`), 굵게(`800`) 설정하여 즉각적인 가성비를 인지시킵니다.
    - 하단에 `#999` 컬러로 '1박 평균' 라벨을 배치합니다.
3.  **우측 (할인 및 총 합계)**:
    - 상단에 할인가 적용 전 원가(취소선)와 **미스터멘션 핑크(`#FF385C`)** 컬러의 할인율을 나란히 배치합니다.
    - 중앙에 최종 결제 금액을 가장 큰 폰트(`24px`, `900`)로 강조합니다.
    - 하단에 기준 박수(예: '29박 기준')를 명시합니다.

---

## 🚀 수정 프롬프트 2: 숙소 상세 페이지 ( [id].vue ) 객실 섹션

**[목표]** 객실 선택 리스트에서 '한달살기 특가' 정보가 일반 가격보다 훨씬 눈에 띄게 구성합니다.

### 🎨 스타일 상세 지침

1.  **할인 배지(Badge) 강조**:
    - `Badge` 컴포넌트나 태그에 **에메랄드 그린(`#008489`)** 배경색을 적용하여 신뢰감과 특가 느낌을 줍니다.
    - 폰트는 화이트, 두께는 매우 굵게(`800`) 설정합니다.
2.  **가격 정보 수직 정렬**:
    - 일반 1박 요금은 상단에 작게 배치하고, 하단에 '한달살기 최저가'를 큰 박스(`price-box`) 형태로 강조합니다.
    - 금액 우측에 '원' 단위를 붙이고 폰트 크기를 차등화합니다.

---

## 🚀 수정 프롬프트 3: 예약 사이드바 (Sticky Sidebar) 실시간 계산

**[목표]** 날짜 선택 시 실시간으로 계산된 할인 내역이 사용자에게 '혜택을 받고 있음'을 확신시키도록 시각화합니다.

### 🎨 스타일 상세 지침

1.  **실시간 할인가(`discount-row`)**:
    - 할인 금액 앞에 마이너스(`-`) 기호를 붙이고 텍스트 색상을 **핑크(`#FF385C`)**로 지정합니다.
    - 할인율(%)을 괄호 안에 넣어 명시적으로 노출합니다.
2.  **최종 합계 정렬**:
    - 구분선(`hr`) 아래의 총 합계 텍스트 두께를 다른 항목보다 훨씬 두껍게 설정하여 최종성(Finality)을 부여합니다.

---

## 🛠️ 공통 스타일 규칙 (Common Guide)

- **Typography**: 프리미엄 느낌을 위해 텍스트 계층(Hierarchy)을 확실히 분리합니다. (강조: `#222`, 부연: `#717171`)
- **Spacing**: 모든 가격 섹션 상단에는 `1px solid #F0F0F0`의 구분선을 넣어 컨텐츠 영역을 명확히 나눕니다.
- **Animation**: 카드 호버(Hover) 시 `translateY(-4px)`와 부드러운 `box-shadow`를 적용하여 인터랙티브한 느낌을 줍니다.
